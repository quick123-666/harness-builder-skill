# Contributing to Harness Builder Skill

感谢你对 Harness Builder Skill 的兴趣！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

1. 检查是否已有类似 issue
2. 提供详细的问题描述
3. 附上复现步骤
4. 说明环境信息（OS, OpenClaw 版本等）

### 提交代码

1. Fork 本仓库
2. 创建特性分支 (git checkout -b feature/amazing-feature)
3. 提交更改 (git commit -m 'feat: add amazing feature')
4. 推送到分支 (git push origin feature/amazing-feature)
5. 开启 Pull Request

### 提交规范

遵循 [Conventional Commits](https://www.conventionalcommits.org/)：

- eat: 新功能
- ix: 修复 bug
- docs: 文档更新
- style: 代码格式（不影响功能）
- efactor: 重构
- 	est: 测试相关
- chore: 构建/工具相关

### 代码审查

- 所有 PR 需要至少 1 个审查
- 确保 CI 通过
- 更新相关文档
